# 仕様

- ユーザ名を入力するとそのユーザの情報を表示することができる
- public activity
- リポジトリ一覧
- 他に出したいものがあれば表示してもよい
- リポジトリの内容を表示することができる
- リポジトリの名前、説明、スターの数、contributors, tag などなど
- 他に出したいものがあれば表示してもよい
- GitHub API を使う https://developer.github.com/v3/
